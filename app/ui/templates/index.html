How it works

  1. Predict the outcome for each group stage match (1 / X / 2)
  2. Predict exact scores for the knockout rounds
  3. Update your predictions before each round's deadline
  4. Earn points for correct predictions:
    • Correct outcome (win/draw/loss): 3 points
    • Correct home team goals: 2 points
    • Correct away team goals: 2 points
  5. Compete on the leaderboard!
